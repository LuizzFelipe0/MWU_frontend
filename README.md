
# Frontend App - Money With You (MWU)

<p><strong>Money With You (MWU)</strong> is the web interface for the MWU personal finance management ecosystem. Developed with <strong>React</strong> and <strong>TypeScript</strong>, it provides a premium, <strong>user experience</strong> to help users manage their financial health with clarity and elegance.</p>

<p>The application acts as a bridge between the user and the MWU FastAPI backend, offering a highly responsive and intuitive dashboard. Users can navigate through their financial categories, transactions, and goals using a clean interface that prioritizes data visualization and ease of use, following Apple's high-quality design standards.</p>

<h2>Key Technologies</h2>
<ul>
  <li>React (UI library)</li>
  <li>TypeScript (static typing & code safety)</li>
  <li>Vite (next-generation build tool)</li>
  <li>Styled Components (CSS-in-JS for modular styling)</li>
  <li>Axios (promise-based HTTP client)</li>
  <li>React Router DOM (declarative routing)</li>
</ul>

<h2>Design Principles</h2>
<ul>
  <li><strong>Scalable Architecture:</strong> Separation of concerns into services, hooks, types, and components.</li>
  <li><strong>Reusable UI Components:</strong> Abstracted CardBox, Button, and Layout components for consistent UI patterns.</li>
  <li><strong>Custom Hooks:</strong> Encapsulated business logic to keep UI components lean and focused.</li>
</ul>

<h2>Requirements</h2>
<ul>
  <li>Node.js 18.0+</li>
  <li>npm or yarn</li>
  <li>Running instance of the <strong>MWU Backend API</strong></li>
</ul>
